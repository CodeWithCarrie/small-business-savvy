## BEFORE YOU CODE

Start planning your page — What elements should be included? The [Planning](./planning.md) file has a list of requirements; use this to write up your content.

1. "Wireframe" your page: grab a pencil and paper and sketch a possible layout. Which parts will go where? What will be big and what will be small? How will images fit into the layout?

2. Once you have the content prepared _and_ the layout planned, _then_ start coding. See [Tips for Beginners](./tips-for-beginners.md) for info & resources that will help you.

## READY TO CODE?

1. Give your page a `<title>` (for the browser tab) in the `<head>`

1. In the `<body>`, use broader-scope elements like `<header>`, `<main>`, `<footer>`, and `<section>` to structure the layout. 
   
1. Use tags like `<ol>`, `<ul>`, `<li>` for lists and `<table>`, `<tr>`, `<th>`, `<td>` for tables if you need a chart of some kind. 
   
1. Use CSS to define default properties for the `<body>` like normal font/size, justification, background color,etc.
   
1. In your HTML file, assign IDs to unique elements, or classes to assign the same style properties to multiple elements. In the CSS file, define the properties you want those to have, like margins and padding to control the structure of the content they will hold. You can even assign different background colors to whole sections.

1. Now, in the CSS file, think about your content and assign specific sizes, colors, justification, etc. to text tags like `<p>` and `<h1>`, `<h2>`, etc. Put some content in your HTML `<section>`s using those tags, and make sure they look how you want. 

1. Create CSS properties for your images (use IDs or classes) like border size/color, max height/width, alignment, etc. and insert them into your content on your HTML page with `<img>` tags and the proper src links.

1. Keep tweaking your page until you've got the basic look you were going for and you've met all the requirements outlined in the [Planning](./planning.md) file. It doesn't have to be fancy, but it should be easily readable and have some personality to it!
