# SMALL BUSINESS SAVVY

A code demo & practice exercise for learners who are new to HTML and CSS

## THE DEMO

After you've forked and cloned this repository, you can open any of the solution pages in the browser on your local machine for reference. Use your browser's Developer Tools to explore the elements and styling.

## THE PRACTICE EXERCISE

When you're ready to create your own unique page, follow the [Instructions](./practice/instructions.md) and refer to the [Tips for Beginners](./practice/tips-for-beginners.md) document for helpful info and resources.

> [!NOTE]
> At any time, you can right-click on the tab of a markdown (.md) file in VS Code and select "Open Preview" for an easy-to-read, compiled view of the document.
